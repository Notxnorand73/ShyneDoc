Getting Started with Shyne

Step 1: Installation

Make sure you have downloaded the appropriate executable for your operating system from the download page.

Step 2: Your First Program

Create a file named `hello.shy` with the following code: 


out("Hello, Shyne World!");

Step 3: Running the Code

Open your terminal or command prompt, navigate to the directory where you saved `hello.shy`, and run the following command: 


shyne hello.shy

You should see "Hello, Shyne World!" printed to your console.
